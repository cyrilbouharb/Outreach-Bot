Steps to Run:

clone repo "git clone https://github.com/cyrilbouharb/Outreach-Bot.git"

go into Outreach-Bot folder

cd into client
run "sudo npm i -force"
run "npm run dev"

open new terminal
cd into server
run "sudo npm update"
run "node index.js"

website can be visited at: http://localhost:3000
